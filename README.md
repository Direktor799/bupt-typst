## 使用方法

进入 [Typst 官网](https://typst.app/) ，并将本模板的文件导入进去，但是你还需要导入 `Times New Roman` 字体，最终你的目录看起来会像是这个样子：

```
- main.typ
- template.typ
- Times New Roman Bold.ttf
- Times New Roman.ttf
```

然后只要修改 `main.typ` 就可以了
